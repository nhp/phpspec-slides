## What is PHPSpec? - based on RSpec - done by Macello Duarte (@_md) & Konstantin Kudryashov (@everzet) - encourage good design - "mimics" the TDD cycle - optimized for descriptiveness

Is it TDD?

no it is a flavor of BDD

Ah so it is like Behat?

not really as Behat is StoryBDD

PhpSpec is SpecBDD

What are the differences to other (test) tools?

Describes individual clases # spec/Example/HelloSpec.php class HelloSpec extends ObjectBehavior { // ... function it_is_saying_hello_for_greeting() { $this->greet->shouldReturn("Hello"); } }
Architectural Design tool
Mimics the "TDD triangle"

TDD Triangle

failing test
passing test
refactor/improve

Spec Triangle

generate spec
generate code
edit code

Conclusion

PhpUnit is a test tool, PhpSpec is a design tool
